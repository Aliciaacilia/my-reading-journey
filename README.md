# 📚 My Reading Journey

Um sistema para gerenciar sua leitura, permitindo cadastrar livros, acompanhar o progresso e organizar sua estante virtual.

---

## 🚀 Funcionalidades

* 📖 Adicionar livros
* 📊 Acompanhar progresso de leitura (em porcentagem)
* 🔄 Atualizar página atual de leitura
* ❌ Remover livros da estante
* 📚 Visualizar todos os livros cadastrados

---

## 🛠️ Tecnologias Utilizadas

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* Node.js
* Express

### Banco de Dados

* MongoDB
* Mongoose

---

## ⚙️ Como executar o projeto

1. Clone o repositório:

```bash
git clone https://github.com/seu-repositorio.git
```

2. Instale as dependências:

```bash
npm install
```

3. Inicie o servidor:

```bash
node index.js
```

4. Abra o arquivo `index.html` no navegador
